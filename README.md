# 💻 Swayam Pawar — Portfolio Website

Welcome to my personal portfolio website built with **React.js, Framer Motion, and modern UI design principles**.  
It showcases my projects, skills, and passion for **Web Development, AI/ML, and Software Engineering**.

🌐 **Live Website:** https://my-portfolio-sp.vercel.app/

---

## 🚀 Features

- 🎨 Modern dark-themed UI/UX design
- ⚡ Smooth animations using Framer Motion
- 💼 Projects, Skills, and Resume sections
- 🧠 Tech-focused portfolio highlighting development expertise
- 📨 Functional contact form using EmailJS
- 📱 Fully responsive design for all devices
- 🚀 Fast deployment with Vercel CI/CD

# ⚙️ Installation & Setup

# 1️⃣ Clone the repository

git clone https://github.com/your-username/portfolio.git

# 2️⃣ Move into project directory

cd portfolio

# 3️⃣ Install dependencies

npm install

# 4️⃣ Create environment file (.env)

VITE_EMAILJS_SERVICE_ID=your_service_id_here
VITE_EMAILJS_TEMPLATE_ID=your_template_id_here
VITE_EMAILJS_PUBLIC_KEY=your_public_key_here

# 5️⃣ Run the development server

npm run dev

# 6️⃣ Open in browser

http://localhost:3000
